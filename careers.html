Possible Careers

Career Description
Front end developer Being a frontend developer consists of a lot of programming, working with others, UI development and problem solving
Data Scientist Being a Data Scientist consists of a lot of database work, data analysis, working with others and optimizing solutions
Network Engineer Being a Network Engineer consists of a lot of hardware, problem solving, working with others and networks and computer systems
Machine Learning Engineer Being a Machine Learning Engineer consists of a lot of artificial intelligence, problem solving, working with data, optimization and working with others
Cybersecurity Specialist Being a Cybersecurity Specialist consists of a lot of Network and computer systems, can entail programming and analytics
Full stack developer Being a Full Stack Developer consists of a lot of programming, UI/UX development, working with data, working with clients, and working with others
Hardware Engineer Being a Hardware Engineer consists of a lot of hardware work, computer and network systems and can be independent
Backend Developer Being a Backend Developer consists of a lot of working with data, protecting data, problem solving, and working with others
Systems Administrator Being a Systems Administrator consists of a lot of working with others, working with clients and stakeholders and problem solving
Database Administrator Being a Database Administrator consists of a lot of working with data and data analytics, working with others and can consist of working with some hardware
Security Analyst Being a Security Analyst consists of a lot of cybersecurity work, some hardware and data analytics
Researcher Being a Researcher consists of a lot of working with others, typically have artificial intelligence involved, can be optimization related, or really is dependent on what topic you research!
Product Manager Being a Product Manager consists of no programming, working with others and working with clients/stakeholders
Ethical Hacker Being an Ethical Hacker consists of a lot of research, working with others, programming and a lot of problem solving
Consultant Being a Consultant consists of a lot of working with others, working with clients and stakeholders and typically no programming
Solutions Architect Being a Solutions Architect consists of a lot of problem solving, programming and working with others
